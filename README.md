# Expense Calculator

Приложение для ведения личных расходов с возможностью добавления транзакций, просмотра истории операций и расчёта общей
суммы затрат. Реализовано с использованием архитектуры MVP, обеспечивает разделение логики, данных и интерфейса.

---

## Демо

Опубликованная версия: **https://thekostvk.github.io/expense-calculator/**

---

## Технологии

- **Vite** — сборка и dev-сервер
- **TypeScript** — логика приложения
- **IndexedDB** — хранение квизов в браузере
- **HTML5 / CSS3** — разметка и стили
- **БЭМ** — методология именования классов

---

## Возможности

- Добавление новых расходов с указанием суммы и категории
- Просмотр списка всех расходов
- Отображение общей суммы расходов
- Автоматический пересчёт данных при изменениях

---

## Архитектура

Использован архитектурный паттерн **MVP (Model–View–Presenter)**.

Архитектура MVP обеспечивает чёткое разделение ответственности:

- **Model** — работа с данными и бизнес-логикой (расходы, баланс, операции)
- **View** — отображение интерфейса и обработка пользовательских действий
- **Presenter** — связывает модель и представление, управляет состоянием и сценариями

## Запуск проекта

1. Склонируйте репозиторий:

```bash
git clone <url-репозитория>
```

2. Установите зависимости:

```
npm install
```

3. Запустите приложение в режиме разработки:

```
npm start
```
